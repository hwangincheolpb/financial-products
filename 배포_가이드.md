# GitHub Pages 배포 가이드

## 1. GitHub 레포지토리 생성

1. GitHub에 로그인
2. 새 레포지토리 생성
   - 레포지토리 이름: `portfolio-proposal-20260123` (또는 원하는 이름)
   - Public 또는 Private 선택
   - README, .gitignore, license는 추가하지 않음 (이미 있음)

## 2. 원격 레포지토리 추가 및 푸시

```bash
cd c:\dev\active-projects\portfolio-proposal-20260123

# 원격 레포지토리 추가 (YOUR_USERNAME을 실제 GitHub 사용자명으로 변경)
git remote add origin https://github.com/YOUR_USERNAME/portfolio-proposal-20260123.git

# 푸시
git push -u origin main
```

## 3. GitHub Pages 설정

1. GitHub 레포지토리 페이지로 이동
2. Settings > Pages 메뉴 클릭
3. Source에서 "Deploy from a branch" 선택
4. Branch: `main`, Folder: `/ (root)` 선택
5. Save 클릭

## 4. 배포 완료 후 URL

배포가 완료되면 (몇 분 소요) 다음 URL로 접속 가능:

```
https://YOUR_USERNAME.github.io/portfolio-proposal-20260123/
```

또는

```
https://YOUR_USERNAME.github.io/portfolio-proposal-20260123/포트폴리오_제안_20260123.html
https://YOUR_USERNAME.github.io/portfolio-proposal-20260123/타임폴리오_롱숏펀드_제안.html
```
